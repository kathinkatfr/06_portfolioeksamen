GRUNDLÆGGENDE WEB

I grundlæggende web lærte jeg om responsivt design, hvordan man bruger det basale html og CSS i VS Code samt metoder og principper indenfor design.


Min computer - site

I denne opgave skulle jeg bruge min viden fra temaet indenfor gestaltlove og brugervenlighed. Jeg skulle følge udleverede wireframes og layoutdiagrammer og bygge mit site op med “mobile-first”.

Sitet er lavet ved brug af CSS herunder media queries som skelner mellem mobilversion og desktopversion og flex og grid til at stille det rigtige layout op med.
